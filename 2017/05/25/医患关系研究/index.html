《医学社会学》课程笔记整理版

推荐读物：帕森斯《社会系统》–奠定功能主义学派

病人和医生——功能主义的视角

  • 社会系统调节器各个组成部分，使之处于相对常恒的范围，保持社会功能的正常发挥。其中某一部分的变化不可避免地在某种程度上影响系统的其它部分。完全从个人出发的或适用于特殊情况的行为可能会被整个社会认为是偏离行为，并因此而受到制裁。
  • 病态是一种功能失调，因为它影响社会系统的稳定性。医学的任务是通过治疗、预防和开发使残疾人独立的技术来抵消病态功能失调。
  • 患病是偏离行为
    • 在一个特定的社会系统中，违反社会规范的任何行动或行为都是偏离行为。
    • 确定偏离行为需要根据社会规范进行社会判断。

病人角色—功能论的观点

  • 人被免除了“正常”的社会责任
  • 病人对自己的疾病状态不负有责任
  • 患病不符合社会需要，因此病人应该具有尝试祛病的愿望（来尽快恢复社会角色）
  • 病人应该寻求专家的帮助并与医生合作

[1] 偏离还有很多其它种类，越轨亦是，但是越轨是得自己负责任，病人则不需要。

医生角色—功能论的观点

  • 医生的作用——社会控制力量（社会角色）
  • 医生的职责——使有病的个人恢复到健康和能够发挥作用的状态，医生通过现代医学手段实现自己的职责。

[1] 公共卫生不仅是专业技术的问题，更是社会问题。

医生的专业特点

  • 技术上的专门性
    • 作为控制力量的资格和能力
    • 健康的象征——文化价值和公信力
  • 感情上的中立性
    • 防止（排斥）主观性
  • 普遍性：一视同仁
  • 职能上的专门性
    • 对控制力量的限制

[1] 可以进一步探讨：专科医生在该专科范围内的控制力量加强 范围还是力度 （38分钟处左右 录音）

医患关系

  • 制度化的角色丛
    • 主要方——病人和医生
    • 次要方——相关他人
  • 一系列对医生角色及其与病人相互影响的行为期待

医患关系的复杂化

  • 从医生——病人关系到医务集体——家庭集体关系的演变正在减少不对称性
  • 健康和疾病定义的扩大、非医学专业的进入（eg. 病人更容易从网上获得信息和资料），医生作为唯一社会控制力量的权利在缩小

[1] 健康的定义：不仅生理状态，还包括心理和社会状态的完好。
[2] 在中国进入商业社会之前，中国的医疗水平还是比现在要低很多的，但是医闹很少。当然，某种程度上也是因为患者对这些知之甚少。

信任关系的基础

  • 医生遵守对医生角色的预期
  • 制度化的预期给医生和病人之间的交往提供稳定性、可预见性和指导
  • 预期防止了病人的被利用

病人——医生关系的不对称性

  • 专业威信
    • 专业训练行医证书
  • 职业权威 保健需求超过供应——卖方垄断
  • 情境造成的依赖性
    • 病人的需求以及自己不能提供满足需求的服务使其必须接受医生的治疗条件

类似孩子与父母的关系

  • 一个人受另一个被社会承认的代表合法控制力量的人的控制
  • 充满感情色彩但控制方必须表现出感情的中立
  • 目标是使使受控制以方便为健全的社会成员

病人——医生关系的特点 心理分析疗法模式

  • 支持—医生对病人提供支持是义务;义务的条件使病人的主观努力
  • 宽容—病人被允许表现出偏离行为而不必受到指责
  • 巧妙利用奖励—控制奖励增加病人对医生的依赖和医生对病人的权威
  • 拒绝互惠—了解病人，但不能被病人了解，以保持优势

对功能论观点的批判

  • 不适用寻求慢性疾病的治疗和寻求预防的医患关系(疾病的性质)，因为慢性病和预防性照顾的目的并非是为了恢复健康
  • 等级障碍和文化障碍妨碍着医患之间的有效交流(病人特点)，医生与较低社会阶层的人的沟通; 一些社会中男性医生诊视女性病人
  • 医生数量的增加减少了医患之间的不对称性，原有关系的继续成了问题(医生特点)

[1] 医生的环境的改变也会改变医患关系。

功能论对批评的反驳

如果要使治疗取得成功，任何治疗关系都是不对称的，有利于医生的，这是与专业人员责任、能力和职务分不开的;不对称分配的确切公式不为人知，随着新知识和新社会条件的出现而变化。

对称性关系类型

  • 竞争性的经济市场—假设：有理性的社会行动者参与交易;交易在经济上有利
  • 交流性网络—假设：信息传递能够平等地获得信息符号所代表的意义;缺乏平等就无法交流
  • 自愿性联盟—参与者在形式上是平等的;一人一篇是权力分配的实质

不对称关系类型

  • 治疗关系—医生的任务是使病人恢复正常，因此要有控制和改变病人行为的力量;因为疾病威胁社会正常秩序 ，因此社会赋予医生控制病人的责任并将其制度化
  • 教学关系—教师的任务是使学生成长为社会期待的人，因此要有控制和改变学生行为的力量;因为无知威胁社会正常秩序，因此社会赋予教师控制学生的责任并将其 制度化

委托关系原则

  • 假定医生是称职的
  • 医生的角色具有专业性 照顾病人是医生的专职 病人生病是暂时的 因而病人角色是被动的 病人赋予医生受委托人角色
  • 预期的病人愿意合作

萨斯—霍伦德的医患关系三模式

依据生理症状的严重程度和可治愈性

  • 主动—被动模式:危重症状，医生单方面的行动、权力和决定
  • 指导—合作模式:急性但非危重，病人能够对疾病作出判断和预期，寻求医生帮助并与之合作(经典模式)
  • 相互参与模式：慢性病症状，病人主动角色，定期看医生;医生“帮助病人自助”

医患互动的理想模式

模式医生角色患者模式模式的临床应用
积极—被动对患者做些事接受者(没有 反应/活动能力)急性休克、昏迷、精神错乱
指导—合作告诉患者做什么合作者(遵从者)急性传染过程
相互参与帮助患者使其 协助自己参与、“合作”，利用专家的协助大多数慢性疾病 、心理咨询

海斯—鲍蒂斯塔模式

  • 基于病人感觉的改变治疗方案的社会过程
  • 两种策略:
    • 说服:1)要求;2)开诚布公;3)建议;4)诱导性提问
    • 反对:1)简单增加;2)额外增加;3简单减少;4)额外减少
  • 医生策略:1)知识压倒;2)开诚布公;3)私人感 情;4)讨价还价

讨价还价后果

  • 病人和医生都满意：关系维持
  • 病人不满意，医生满意：病人单方面中断关系
  • 病人满意，医生不满意：医生单方面中断关系
  • 病人和医生都不满意：关系结束

讨价还价过程

  1. 病人感到治疗方案不充分
  2. 不稳定的服从或不服从
  3. 医生觉察或未觉察病人的行动
  4. 医生觉察后的协商重新确定关系
    1. 继续保持关系
    2. 病人愿意结束关系
    3. 医生愿意结束关系
    4. 双方愿意结束关系

海斯—鲍蒂斯塔模式的局限

  • 没有普遍适用的验证
  • 只考虑病人的感觉，没有医生的感觉，也没有考虑医 患二者感觉的协调
  • 没有介绍测量感觉的指标
  • 增加了对病人不服从和医生控制过程的了解

医生对病人了解程度的研究 斯图尔特和巴克

  • 医生对病人问题的了解程度和联系
  • 医生对病人问题的反应程度和联系
  • 医生对病人问题的了解和反映之间的联系

研究发现

  • 医生对病人的了解局限于痛苦、不舒服、忧虑和日常生活失调(46-62% )，对与病人的社会问题知之甚少( 23%)
  • 医生对病人问题的反应只在不舒服方面是充分的，而且对病人各方面问题的反应互不相连;反应最不充分的是 日常生活失调。
  • 当医生觉察到病人的问题后，会作出反应而不考虑问题的特出性质;了解和反应之间不密切的关系表明医生对病人的问题很不敏感。
  • 对医患关系的意义——医生不关心病人的非医学问题，无疑降低效率，也预示着医生既不可能重新确定治疗方 案以安抚不愉快的病人，治疗过程的可协商性是十分有限的。

关于医患交流的研究

  • 问题:如果关系不对称，从理论上讲交流是不可能的
  • 方法:对医疗交谈的分析应包括提出和回答问题的人，打断谈话的人，误解和被误解者，引起笑声者等等。
  • 结果:男性医生打断谈话病人谈话次数弊病人打断他谈话次数多一倍;黑人病人谈话更容易被打断;而病人打断女医生比女医生打断病人的话更经常;医生最可能先提问，而病人最可能回答医生的提问。

对医患关系的意义

医患关系中存在不对称性和对称性，医生并不总是处于支配地位，尽管在某些方面确实占上风。

对联盟形成的分析 科恩

  • 医生运用特殊策略促进与老年病人的交流和病人的服从，通过家属，家属的作用就像父母之对于孩子 ，把老年病人当作非成年人对待
  • 家属扮演解释者、协商者和照顾者角色，对医患关系发生影响
  • 三人联盟的形成，其中是两人联合影响一人

医患关系的研究方法问题

  • 理论取向:功能主义强调角色预期和不对称关系的必要性;交换(霍斯曼)强调交换观念，理性的计算得失来决定交往;冲突(韦斯特)强调医 生和病人的不同利益和手段，以及协商结果的必要。
  • 学科取向:社会行为科学关心的是医患之间的交流过程;临床医生的重点是过程结果。
  • 一般概念模式：(伊纽伊和卡特)

有待研究的问题

  • 医患关系的多个(所有)方面(有效沟通、文化差异、病人遵从等)
  • 互动质量的影响因素 (包括地位、教育、职业、性别和权威等方面的差异)
  • 影响因素与结果是一般相关还是因果关系，需要长期追踪研究
