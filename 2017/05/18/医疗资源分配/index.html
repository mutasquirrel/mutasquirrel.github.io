问题

  • 卫生保健支出快速增长（全球性的问题）
  • 卫生保健覆盖不足
  • 公众满意程度较低
  • 昂贵和复杂的管理要求

[1] 做研究的时候，这个“资源” 要定义清楚，什么样的资源，是医院、设备、医生还是现代保险等等

不同的改革建议

  • 私人市场保险 （e.g. 美国）
  • 雇主支持的卫生保险
  • 集中或单一卫生保健支付模式

中国卫生资源结构

  • 社会医疗保险（个人和企业缴费）
    • 企业雇员医疗保险
    • 城市居民医疗保险
    • 新型农村合作医疗保险
  • 公费医疗（财政支出）
    • 公务员
    • 事业单位雇员
  • 医疗救助计划（财政支出）
  • 商业医疗保险（个人缴费）

卫生保健危机的社会意义

  • 重治疗预防导致更高的成本
  • 医疗资源分配不公平影响人口健康状况
  • 卫生保健作为公民的一个权利

不同医疗保险计划面对不同的问题

  • 全民医疗保险（基于公民权利）
    • 资源如何有效地分配
    • 医疗保健提供方式
    • 医疗保健获得方式
  • 分散和个人医疗保险（基于自由市场原则）
    • 社会经济地位决定医疗资源的获得
    • 就业状况决定医疗资源的获得

影响医疗成本上涨的因素

  • 人口数量的增长，人口的老化
  • 组织—医院“内置”成本的增加；公司化 和营利化的医院组织出现
  • 技术—复杂的医疗技术
  • 机制—第三方付费
  • 文化—医疗服务提供者和公众伦理观念

经济合理的卫生保健制度

  • 人人都能获得基本卫生保健（公平）
  • 有办法限制使用无效或勉强有效，以及虽然有效但过分昂贵的医疗程序（效率）
  • 社会和个人层面关于卫生保健优先项目界定的共识

[1] 中国点滴滥用现象：吃药、肌肉注射、点滴这三种方式，安全性递减，医效递增，价格递增，但是中国点滴滥用现象十分严重。
背后的逻辑：患者没有医学常识，医院被裹挟，混乱的市场

合理卫生保健制度的价值基础

  • 每人（患者、医生、医院和卫生管理者）都有自治权和选择的自由
  • 医学进步是无限的，任何疾病最终都会被治愈的
  • 人类健康寄希望于医学和卫生保健质量和不断改进和创新的技术
    （Callahan，1997）

追求与挑战

  • 卫生保健的追求：
    • 无限的医学进步
    • 最大化的选择
    • 完美的健康
    • 利润和收入
  • 费用控制的挑战
    • 需求的增长
    • 新型技术的爆炸性增加

医学进步的麻烦

  • 其经济机会是内在无限的
  • 其制造困惑和对立面的能力是内在无限的

价值观的改变

  • 区别普及现有医学知识／技能和推动医学前沿寻找挽救，改善生命
  • 认识到我们已经是人类历史中最健康和最长寿的人了
  • 认识到自主权、选择权和高质量的价值追求的减少不会对实际健康呦任何严重的损害
  • 认识到卫生保健成为利润扩展和个人致富来源已经使社会回报在减少

保健分配的问题

  • 医学的合理性问题—制定能够适应所有人的合理分配规定在医学上是不可能的
  • 伦理和政治问题—医生在拒绝为某个别人时提供某项治疗时会不舒服；必须有统一的资源渠道或固定的预算
  • 长远看并不省钱—经济的压力促使提供者扩展其他不涉及分配的服务

费用节制和质量保证面临的问题

  • 在许多卫生保健系统中，都曾有过提供不必要医疗服务和没提供必要医疗服务的情况
  • 单方面强调减少不必要的服务，也会同时减少必要的服务
  • 从绝对意义上讲，卫生保健的一般质量是很低的，所以应该充分利用每一个机会来提高质量
  • 人口中的所有亚群体没有得到相同质量的医疗服务
  • 医生的行医表现，个体层面差异很大
  • 病人和医生对医疗质量的评价并不总是一致

卫生改革的原则

经济学不是医疗保健改革的唯一原则

  • 伦理道德原则
  • 机构与协调机制的理想特性
  • 理想的配置比例

伦理道德原则

  • 扩大个人自主权，减少政府干预程度
  • 帮助处于困境和劣势地位的人

机构与协调机制的理想特性

  • 竞争性、无国家所有和垄断；不同所有制和协调机制间充分的竞争
  • 有效激励，建立提高效率的所有制和控制形式
  • 政府角色，提高法律框架，监管非国有机构，通过最后的救助与保险
  • 透明性，公民明确服务与融资赋税之间的关系；公开，公众知晓的讨论
  • 实践充分，足够的时间让卫生部门的新机构得到发展，让公众学会适应

卫生部门的普遍特征

  • 健康的价值：健康是无价的，但是资源是稀缺的，这个领域是“市场失灵”的领域
  • 平等获取的原则：基本需要，不仅是生物和物理，还有社会价值，同体面生活密不可分
  • 不确定性和保险需求
  • 信息不对称
  • 选择： 逆向选择：最愿意加入保险的人是成本最大的人
  • 风险选择：保险越多或越全面，投保人避免危险和减少损失的激励就会越弱
  • 供应方权力和垄断
  • 病人的无助：不论社会制度和经济机理如何，医生都对病人拥有权力
  • 不断增加的费用（只可能降低费用增长的速度，不可能降低费用）
