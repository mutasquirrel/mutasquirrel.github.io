    Black Report

    • 1980年，由DouglasBlack爵士领导的专家委员会受英国卫生与社会保障部的委托发布的报告(从1977年开始)
    • 主要内容:
      • 健康不平等广泛存在
      • 主要原因–经济不平等
      • 健康不平等状况有扩大的趋势
    • 主要影响:
      • OECD 国家和WHO对健康不平等的关注和政策
      • 关于健康不平等的研究

    [1] 《Black Report 1980》全文

    健康不平等的解释

    • 生物学因素
      • 基因、天生残疾等等
      • 健康选择理论，身体健康的差导致各种机会（教育、工作等等）被剥夺
    • 物质资源因素
      • 资源对健康状况的影响
      • eg.留守儿童和城里孩子
    • 社会影响因素
      • 生活方式、住房、饮食、受教育资源的享用等等
    • 文化、环境和阶级之间的互动及其对健康的影响
    • 社会性别(gender)作用解释的困惑
      • 区分：生物性别(sex) 其实这方面也有区别

    [1] 健康状况是一种结果，很多因素作用的结果
    [2] 关于“痛经”的思考，痛经某种程度上不算疾病，它是医疗扩大化的结果，人们的耐痛力下降
    [3] 中国的种族之间的健康差异是很大的，但因为是敏感话题，鲜有人研究。可参考数据：人口统计年鉴–人口平均预期寿命

    健康是否具有个人和社会财产属性?

    • 主观感觉
      • 关于健康和疾病的信念反映着关于道德、责任、和因果的文化模式
        因为，比如说，不同的文化模式下，“病”的定义都不同
      • 人们对不健康挑战的回应，或试图维持健康的努力深受社会结构的影响
        例如，城市老人的广场舞的兴起；老年人衰退型的症状其实不算是病但却老是去治疗，等等现象
    • 客观测量
      • 死亡率和发病率的比较发现一个社会之中和社会与社会之间的明显差异
    • 争论议题
      • 物质环境和资源差异?
      • 社会群体之间类型差异是如何形成的?
      • 什么社会和物质力量如何型塑了不同的健康类型?

    议题之一:社会关系与健康

    • 社会整合与社会变迁的作用(Durkheim)与依恋理论(John Bowlby)
    • 宏观影响——社会经济因素、文化政治因素、社会变迁性质
    • 中观影响——社会网络特征、社会互动
    • 微观影响——社会支持、自我评价

    涂尔干(Emile Durkheim)的社会整合

    • 个人生理状况是社会动力作用的结果
    • 自杀率与一个群体的社会整合水平相关
    • 失范型自杀与大规模经济和政治性社会危机相关，与快速的社会变迁和动荡相关
    • 社会快速变化解除了对原有价值观、信仰和常态的规范，无法驾驭或引导个人的理想

    鲍比(John Bowlby)的亲附理论(Attachment Theory)

    • 人类需要建立亲密的纽带关系
    • 可依附的人可以为依附者提供一个的安全基地令其探索前行(母亲与婴儿)
    • 在一个更大体系中，安全依附令情感纽带和安全感得以维 持
    • 建立亲密关系的能力受复杂因素的影响，包括依附、分离 、丢失、重新依附等
